# Reviewer Response Support Files

This folder collects concise Markdown summaries used to update the manuscript,
supporting information, and point-by-point reviewer response.

Included topics:

- reviewer-requested chemometric and neural baselines
- SHERLOC fine-tuning and transfer protocol
- confidence-threshold and rejection analysis
- MST-focused tuning notes
- data augmentation and materialized dataset rationale

Machine-readable experiment outputs are stored under `results/`, while source
metadata and training-ready tables are stored under `data/`.
